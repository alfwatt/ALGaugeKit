
# GaugeKit.framework

## Real-Time Gauge Views for iOS

- Indicator, with Styles
    - Text
    - Vertical Bar
    - Horizontal Bar
    - Square
    - Circle
    - Ring
    - Pie
    - Dial

- Timelines, including
    - Time Series
    - Histogram
